This is the code and mathematica file for the manuscript "[Quasi epigenetic equilibrium and it's implications for evolutionary rescue](https://www.biorxiv.org/content/10.1101/2025.08.27.672696v1.abstract)"

Running the bash scripts in the "Stochastic Simulations" folder will generate all the data stored in "DataFiles" required to run the code in "FinalFigures.ipynb" and produce the figures in the "Figures" folder. 

The mathematica file "PopEpi_Mathematica.nb" has the analytical calculations for the pre-change model.
